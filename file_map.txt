Phase 1 — Data distillation
handtracking/topology.py — 10 MediaPipe indices (wrist, 4 MCPs, 5 tips; middle PIP omitted).
handtracking/teacher.py — MediaPipe Hands teacher.
handtracking/geometry.py — Letterboxing to 160².
handtracking/augmentations.py — Motion blur, 180° rotation + keypoints, cutout.
handtracking/distill_freihand.py — JSONL manifest writer.
handtracking/verify_samples.py — samples.png grid.
Demo data under data/demo_images/ (TF OD test image ×10); data/distilled/manifest.jsonl and data/distilled/samples.png generated successfully.
Phase 2 — Model + loss + check
handtracking/models/mobilenet_v4_conv_small.py — Conv-only small stack (UIB-style blocks).
handtracking/models/hand_simcc.py — SimCC head → simcc_x / simcc_y each [B, 10, 320].
handtracking/losses.py — Label-smoothed SimCC CE + Adaptive Wing on soft-argmax coords.
handtracking/verify_forward.py — Confirms (1, 10, 320) per branch.
Phase 3 — Train, export, size gate
handtracking/dataset.py — Manifest dataset + augmentations.
handtracking/train.py — AdamW + ExponentialLR; optional QAT (saves .fp32.pt before convert for ONNX).
handtracking/export_onnx.py — Fixed [1, 3, 160, 160] ONNX.
handtracking/phase3_verify.py — Size check (< 5MB fallback on ONNX when NCNN tools are missing).
scripts/convert_ncnn.sh — onnx2ncnn + ncnnoptimize when installed.
scripts/onnx_to_tflite.sh — Android path placeholder.
Checkpoint and ONNX were produced locally (checkpoints/hand_simcc.pt, models/hand_simcc.onnx ~3.86 MiB < 5MB).

Phase 4 — C++
cpp/main.cpp — Thread A: V4L2 VideoCapture + letterbox; Thread B: NCNN + inference_time_ms log; omp_set_num_threads(4); SimCC decode + 1€ filters.
cpp/OneEuroFilter.hpp — 1€ filter.
cpp/CMakeLists.txt — OpenCV + optional ncnn; OpenMP.
Built without ncnn here: binary exits 2 with a clear message; with -Dncnn_DIR=... and converted .param/.bin, it runs the full pipeline.
handtracking/bench_torch.py — CPU PyTorch timing when NCNN is not available.
Phase 5 — 3D / PnP
handtracking/pnp_kinematics.py — Normal (index\_MCP - wrist) × (pinky\_MCP - wrist), cv2.solvePnP on wrist + 4 MCPs, demo splay/curl logging (--demo).
Commands (from repo root, PYTHONPATH=. or pip install -e . if you add packaging later)

PYTHONPATH=. python3 -m handtracking.verify_samples
PYTHONPATH=. python3 -m handtracking.verify_forward

cd cpp/build && cmake .. -Dncnn_DIR=/path/to/ncnn/lib/cmake/ncnn && cmake --build .
PYTHONPATH=. python3 -m handtracking.pnp_kinematics --demo
Note: .gitignore ignores checkpoints, ONNX, and NCNN blobs so large artifacts are not committed by default. FreiHAND: point --data-root at the dataset root (with training/rgb or your image folder).
